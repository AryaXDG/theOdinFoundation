first lets learn about the basic syntex

Basic Syntax:


    selector {
        property : value;
    }

Types of selectors:

Universal(*):

This selector selects every type of element and apply rules that are written in the form of property and value pair

Type Selector(div/p/h2):

You can just put the name of the type as a selector and then follow the normal syntax

Class Selector

We made a class here called highlight to make this this colour, this only applies to the element under this class

This text is normal.

ID selectors

This a unique identifier which is kind of like a class but, The major difference between classes and IDs is that an element can only have one ID

Grouping Selectors

we just write different elements and group them by comma (,)
